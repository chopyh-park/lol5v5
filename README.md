# LoL 5v5 매칭 도구

`matcher.py`는 10인의 플레이어를 입력받아 DFS로 팀 조합을 전수조사하고, 라인 단위 MMR·오프롤 보정·금지 매치업을 반영해 가장 공정한 팀을 추천하는 스크립트입니다.

## 핵심 특징
- 티어→MMR 선형 매핑과 전문/범용 λ 테이블을 활용한 라인별 점수 계산【F:matcher.py†L63-L118】【F:matcher.py†L135-L171】
- 팀 평균 티어에 따라 승률 로지스틱 스케일을 조정하고, cost 함수에서 승률 편차에 높은 가중치를 부여【F:matcher.py†L120-L132】【F:matcher.py†L223-L249】
- 60% 초과 승률 후보를 강제로 제거하는 캡을 추가해 과도한 승률 격차를 사전에 차단【F:matcher.py†L288-L301】【F:matcher.py†L482-L499】
- 윤영 Top/Jg 두 경로를 모두 탐색해 더 균형 잡힌 후보를 제시하고, per-lane 편차를 강하게 페널티하는 새 cost 설계를 적용【F:matcher.py†L334-L367】【F:matcher.py†L468-L510】
- 대칭 매치 제거, 금지 매치업, NOT_SAME_TEAM 제한, 역할 고정/교대 로직을 지원【F:matcher.py†L251-L286】【F:matcher.py†L482-L510】

## 실행 방법
```bash
python matcher.py
```

`rules.yaml`을 같은 경로에 두면 플레이어별 목표 티어, 라인별 오버라이드, 금지 매치업, 동일 팀 금지 목록을 덮어쓸 수 있습니다.

## 사용 시 주의점
- 승률 캡(기본 60%) 때문에 모든 결과가 제거될 수 있습니다. 이때는 원본 검색 결과를 사용해 빈 결과를 방지합니다.【F:matcher.py†L288-L301】
- 표준 입력 대신 코드 상의 `PLAYERS_TEXT`를 수정해 플레이어 목록을 관리합니다. 필요하면 CSV 파서 등을 추가로 붙여서 사용하세요.【F:matcher.py†L303-L327】
